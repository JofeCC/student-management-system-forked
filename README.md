# Student management System

## .Net Core

- List the templates
    ```shell
    dotnet new list
    ```
- Generate a new WebAPI
    ```shell
    dotnet new webapi -n {NAME}
    ```
- Run the API
    ```shell
    // go to project folder
    dotnet run
    ```
- Install EF Core CLI
    ```shell
    dotnet tool install --global dotnet-ef
    ```
